This project connects to the Microsoft SQL Database modify and retrieve data from the consultant management system. 


1) The sirmoDBMSnew.py uses the built in pyodbc library to connect to the MS SQL database. It implements the following functions:
connectDB
addNewConsultant
removeConsultant
getMinHourlyRateAVG
getConsultantInfo
getConsultantRolePay
connectDBClose


2) The SirmoDBMSTest.py is a unit testing enviroment
